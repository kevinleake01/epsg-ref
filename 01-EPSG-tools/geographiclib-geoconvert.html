Using GeoConvert

GeoConvert is a tool from GeographicLib that converts between WGS84 Lat-Long, UTM/UPS, and MGRS.
To get started, we will now install GeographicLib:

$ git clone https://github.com/geographiclib/geographiclib
$ cd geographiclib
$ mkdir BUILD
$ cd BUILD
$ cmake ..
$ make
$ cd tools

We are now ready to use GeoConvert. Here are some examples from the Royal Observatory
in Greenwich, located at EPSG:4326: 50.938357 -1.472726

To convert to UTM: (EPSG:32630: 607308 5644081)
$ ./GeoConvert -u --input-string "50.938357 -1.472726"
30n 607308 5644081

To convert to MGRS:
$ ./GeoConvert -m --input-string "50.938357 -1.472726"
30UXB0730744080

To convert back to Lat-Long:
$ ./GeoConvert -g --input-string "30n 607308 5644081"
50.93836 -1.47272

$ ./GeoConvert -g --input-string "30UXB0730744080"
50.93836 -1.47273
