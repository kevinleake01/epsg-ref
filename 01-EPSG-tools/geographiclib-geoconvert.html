Using GeoConvert

GeoConvert is a tool from GeographicLib that converts between WGS84 Lat-Long, UTM/UPS, and MGRS.
To get started, we will now install GeographicLib:

$ git clone https://github.com/geographiclib/geographiclib
$ cd geographiclib
$ mkdir BUILD
$ cd BUILD
$ cmake ..
$ make
$ cd tools

We are now ready to use GeoConvert. Here are some examples from the Royal Observatory
in Greenwich, located at EPSG:4326: 51.477811 -0.001475

To convert to UTM: (EPSG:32630: 708216 5707226)
$ ./GeoConvert -u --input-string "51.477811 -0.001475"
30n 708216 5707226

To convert to MGRS:
$ ./GeoConvert -m --input-string "51.477811 -0.001475"
30UYC0821507225

To convert back to Lat-Long:
$ ./GeoConvert -g --input-string "30n 708216 5707226"
51.47781 -0.00147

$ ./GeoConvert -g --input-string "30UYC0821507225"
51.47781 -0.00148
